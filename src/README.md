# Mergington High School Activities

A super simple website application that allows students to browse extracurricular activities, and teachers to register/unregister students.

## Features

- View all available extracurricular activities
- Filter activities by category, day, time, and difficulty
- Search activities by keyword
- Share activities with a copy/share link
- Teacher login to register students for activities
- Teacher ability to unregister students from activities
